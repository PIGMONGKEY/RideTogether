# 같이타요

## 개요
- 신한대학교 동두천 캠퍼스 학생들을 위한 카풀 및 택시 동승 서비스

## 사용 기술 및 개발 환경
- Language : Java, PHP
- Server : Apache

- DB : MySQL
- Tool : Android Studio
- OS : MacOS, Kali Linux
